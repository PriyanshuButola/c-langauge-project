# Contact Management System

A simple command-line Contact Management System written in C. Built as a practice project to apply core C concepts — structs, arrays, pointers, and file I/O — in a real, working application.

## Features

- Add new contacts (name, phone, email)
- View all saved contacts
- Search for a contact
- Edit an existing contact
- Delete a contact
- Save contacts to a file and load them back on startup (persistent storage)

## Concepts Practiced

- Structs and arrays of structs
- Pass by value vs. pass by pointer
- File handling (`fopen`, `fprintf`, `fscanf`, `fclose`)
- String handling (`fgets`, `strcpy`, `strcmp`)
- Modular code using functions
